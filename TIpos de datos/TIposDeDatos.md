# Tipos de datos
En PHP tenemos los siguientes tipos de datos:

- String    (cadena de caracteres)
- Integer   (un entero)
- Float     (entero con decimal)
- Boolean   (booleanos)
- Array     (arreglos)
- Object    (objetos)
- NULL      (nula)
- Resource

Para saber que tipo de datos es una variables se utiliza lo siguiente:

`var_dump($var);`
